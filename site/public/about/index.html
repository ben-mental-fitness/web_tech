About CoVision

CoVision was created to help clear the fog about coronavirus and help people see the facts clearly. There has been so much disinformation present at all levels of society about this pandemic, which is detrimental to our response and recovery effots. It is important that we understand how this pandemic has affected the world so that we can learn for the future.

How our platform works

Each story is a different visualisation representing some aspect of the pandemic. To ensure correctness & transparency each data source is embedded into the visualisation itself so anyone can check its validity and then challenge our platform if they believe it is not accurate.
